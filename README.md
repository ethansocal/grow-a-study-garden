# Grow a Garden -- Team Early: Ethan, Andrew, Ryan, Louis, Yves

## Installation

### Prerequisites

1. Download Bun from <https://bun.com>
2. Download Godot 4.6 from <https://godotengine.org/download>

### Running the program

#### Next.js

1. Navigate to `./website`
2. Run `bun install`
3. Run `bun run dev`

#### Godot

1. Open `./godot` in Godot
2. Click run project
